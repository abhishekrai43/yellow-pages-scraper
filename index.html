# yellow-pages-scraper Python Selenium based scraper which scrapes the details of businesses and outputs a csv in the same folder. Gets Name, Category, Phone Number, Address and locality. **REQUIREMENTS** `pip3 install selenium` **INSTALLATION** simply run `git clone https://github.com/abhishekrai43/yellow-pages-scraper` `cd yellow-pages-scraper` **RUN** `python3 yellow-pages.py`
